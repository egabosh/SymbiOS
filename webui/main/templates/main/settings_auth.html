About

Authelia is an open-source authentication server that integrates with Traefik as a forward-auth middleware. Once enabled:

  • All web services behind Traefik require login
  • Users authenticate against the local LDAP directory
  • Optional 2FA (email or TOTP) adds a second factor

Manage users in Users and groups in Groups. Without 2FA, only a username/password is required.
