         About deSEC
      

      

        

          deSEC is a free Dynamic DNS provider. Your server
          periodically (cron, every 5 min) updates the DNS A/AAAA
          records for your hostname so it always points to your current public
          IP — even when your ISP changes it.
        

        
    
          
  • Get an API Token at desec.io
    • 
          
  • Hostname: myserver.dedyn.io (subdomain only)
    • 
          
  • Domain is auto-created if it doesn't exist yet
    • 
          
  • DDNS update runs every 5 minutes via cron
    • 
          
  • Updates are logged in System Logs
    • 
        

        

          
          Need help with port forwarding? See the guide at
          Settings → Port Forwarding Guide.
        

      

    


    
    

      

         About self-managed DNS
      

      

        

          Use your own domain and manage DNS externally (e.g. at your registrar
          or a DNS provider like Cloudflare, BIND, etc.).
        

        
    
          
  • Enter your domain (e.g. example.com)
    • 
          
  • All services will use subdomains (symbios.*, auth.*)
    • 
          
  • You must manage DNS records yourself
    • 
          
  • Wildcard CNAME recommended for ease of use
    • 
        

        

          
          Need help with port forwarding? See the guide at
          Settings → Port Forwarding Guide.
