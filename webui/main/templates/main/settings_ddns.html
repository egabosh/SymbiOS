About

This configures deSEC Dynamic DNS for your server. Changes are applied automatically via the Config Daemon.

  • API Key from desec.io
  • Host must be your subdomain (e.g. myserver.dedyn.io)
  • Domain will be auto-created if it doesn't exist
  • DDNS update runs every 30 minutes via cron
  • Logs visible in System Logs

Dynamic DNS is required to make your server reachable over the internet when your ISP does not provide a static IP address. It maps a fixed hostname to your changing public IP, so you can always access your server via myserver.dedyn.io. With private consumer internet connections, IPv4 is often behind Carrier-Grade NAT (CG-NAT) which makes incoming connections impossible without a tunnel/VPN — in that case IPv6 is the only way to reach your server directly. With IPv6, each device typically has a public, globally routable address natively.
