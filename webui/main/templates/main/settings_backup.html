About

Configure an SSH/SFTP server for automated backups. The server must be reachable via SSH with key-based authentication.

  • The WebUI's SSH key is used for authentication
  • Test the connection before saving
  • The remote directory must already exist
  • Backups are stored as timestamped files

Ensure the SSH key is authorized on the remote host (see SSH Keys).
