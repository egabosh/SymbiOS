Setup /home Partition
Select an unused disk or partition to format and mount as /home. Existing data from the current /home will be copied automatically.
The partition will be encrypted with LUKS. You will need to enter the passphrase at each boot via this WebUI.
If you forget this passphrase, your data is permanently unrecoverable. Write it down and store it somewhere safe.
Passphrases do not match.
About

Move /home to a dedicated disk for better performance and more storage. Supports optional LUKS encryption.

How it works:
  1. Select an unused disk
  2. Optionally enable encryption
  3. Click "Setup Disk"
  4. Data is copied automatically
  5. /home is remounted on the new disk
Important: Back up your data before proceeding. The target device will be completely erased.
